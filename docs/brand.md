# Quad visual direction

## Current lock

Quad uses a light editorial cherry-blossom system:

- Paper: `#fff8fb`
- Panel: `#fff0f6`
- Strong panel: `#ffe1ee`
- Ink: `#27131c`
- Muted text: `#6f3f55`
- Hairline: `#e998b9`
- Accent: `#c21f63`
- Dark accent: `#8f174b`

## Motion

The background uses ASCII blossom petals, ribbon text, and hover-triggered ASCII bursts. Motion should stay decorative and must not block app controls. Respect `prefers-reduced-motion`.

## Copy

Product UI uses sentence case. Keep the assistant/chat personality separate from interface labels.

## Interaction feel

The app should feel like an editorial command surface, not a landing page. Keep the chat, live logs, findings, approval previews, and debug drawer usable first; decorative ASCII should support the product instead of competing with it.
